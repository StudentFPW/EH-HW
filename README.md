# Практическое задание (HW)

```shell
1. Фаззер файлов, принимает на вход расширения файлов (например, .txt, .php и т.д.) и записывает найденные файлы с этими расширениями в file.txt

2. В качестве списка файлов и директорий, которые должны проверяться на сайте, возьмите:
    common-files.txt

3. В качестве сайта для тестирования:
    https://www.hackthissite.org/

4. В качестве ответа необходимо предоставить:
    1. Код программы.
    2. Скриншоты из командной строки с вводом необходимых значений.
    3. Файл с отфильтрованными результатами сканирования.
    4. Расширения вводятся через input в любом количестве интерактивно
    5. При работе с файлами студент закрывает дескриптор файла после внесения/чтения данных методом .close()

Условия оценивания:
    Программа позволяет интерактивно вводить список расширений или фильтров.
    Программа сохраняет отфильтрованные ссылки на файлы/директории в отдельный файл file.txt
    Использована библиотека requests.
```
